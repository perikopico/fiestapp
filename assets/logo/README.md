# Logos y assets de QuePlan

## Logo de la barra superior (AppBar)

Para que el logo aparezca en la barra superior del dashboard:

1. **Ruta:** `assets/logo/queplan_logo.png`
2. **Nombre del archivo:** `queplan_logo.png`
3. **Formato:** PNG (transparente o con fondo, según prefieras).

Coloca tu imagen en esta carpeta con ese nombre. Si el archivo no existe, se mostrará el texto "QuePlan" como respaldo.

## Otros assets

- `icono.png` — Icono anterior de la app.
- `icono2.png` — Icono actual de la app (launcher). Usado por `flutter_launcher_icons`.
- `queplan_logo.png` — Logo en el AppBar del dashboard (incluido en `pubspec.yaml` → `assets/logo/`).
